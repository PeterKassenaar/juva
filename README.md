## During the training
- Three types of state management:
  - 1. Classic, inside one component using `useState`: https://react.dev/reference/react/useState
  - 2. Context API: if you need to share state between components within the __same parent component tree__, use the Context API with `createContext` and `useContext`: https://react.dev/reference/react/createContext and https://react.dev/reference/react/useContext.
  - 3. Redux: if you need to share state between components across different parent component trees, use Redux. This is complex, but gives you a lot of flexibility: https://redux.js.org/introduction/getting-started
